# 📊 Superstore Sales & Profit Analysis Dashboard

## 📌 Project Overview

This project is an interactive **Power BI dashboard** built using the Sample Superstore dataset. The report provides business insights through three dashboard pages focusing on executive performance, sales analysis, and profitability analysis.

---

## 🛠️ Tools & Technologies

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel

---

## 📈 Dashboard Pages

### 📌 Executive Dashboard
- Business KPIs
- Sales & Profit Overview
- Regional Performance
- Interactive Slicers
- Executive Insights

### 📌 Sales Analysis
- Monthly Sales Trend
- Sales by Region & Segment
- Sales by Category
- Top Products
- Top Sub-Categories

### 📌 Profitability Analysis
- Monthly Profit Trend
- Profit by Region & Segment
- Profit by Category
- Profit by Sub-Category
- Bottom 5 Loss-Making Products

---

## 📊 Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Profit Margin
- Total Orders
- Average Order Value
